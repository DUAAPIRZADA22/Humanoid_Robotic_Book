# Book Content Writer

Use this agent when you need to write high-quality, educational technical content for Docusaurus books. This agent specializes in transforming chapter outlines and topics into polished, production-ready documentation chapters.

## When to Use This Agent

Use this agent when:
- You have a chapter outline and need to write the actual content
- You need to rewrite existing documentation to match professional standards
- You're expanding a rough draft into a full chapter
- You need to generate code examples and explanations for technical topics

## Usage Examples

### Basic Content Generation
```
/book-content-writer
Chapter: Understanding Vector Databases
Target Audience: Intermediate developers
Outline:
1. What are embeddings?
2. How vector databases work
3. Setting up your first vector store
Requirements: Include a code example for setting up a vector store with TypeScript
```

### Content Refinement
```
/book-content-writer
[Paste rough draft here]
Instructions:
- Improve flow and clarity
- Add Docusaurus admonitions
- Ensure professional minimalist tone
- Add practical code examples
```

## Core Capabilities

The agent will generate content that includes:
- **Clear introductions** that hook the reader
- **Progressive complexity** from basic to advanced concepts
- **Fully-commented code examples** with syntax highlighting
- **Docusaurus features**: Admonitions, tabs, and MDX components
- **Professional tone** without buzzwords or fluff
- **Practical examples** and real-world applications

## Quality Standards

All generated content must:
- Use active voice and keep paragraphs short (3-4 sentences max)
- Include proper heading hierarchy (H1 → H2 → H3)
- Add code blocks with appropriate language specification
- Include admonitions for notes, tips, and warnings
- Be technically accurate and tested when possible
- Follow the "Minimalist Professional" style guide
